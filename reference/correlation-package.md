# correlation: Methods for correlation analysis

Lightweight package for computing different kinds of correlations, such
as partial correlations, Bayesian correlations, multilevel correlations,
polychoric correlations, biweight correlations, distance correlations
and more. Part of the 'easystats' ecosystem.

References: Makowski et al. (2020)
[doi:10.21105/joss.02306](https://doi.org/10.21105/joss.02306) .
